Bienvenue au fichier d'aide de Métamorphose2 !
Cette fenêtre peut être gardée ouverte pendant que le logiciel est utilisé.

Comment utiliser Métamorphose-2 :

  • Dans le panneau Ramasseur, cliquer le bouton Parcourir pour sélectionner un chemin d'accès. Il est aussi possible d'entrer ou de coller un chemin puis de cliquer le bouton Rafraîchir. Le contenu de ce chemin sera alors affiché.

  • Par défaut, lorsqu'un chemin est chargé tout son contenu est sélectionné. Pour (dé)sélectionner des fichiers ou dossiers il suffit de les cliquer. Toutes sélections utiliseront les mêmes règles pour être renommées. Si des dossiers sont sélectionnés, leurs contenu ne sera pas changé.

  • Dans le panneau - Renommer -, choisir les opérations qui vous conviennent en les double-cliquant ou en glissant-déposant. Les paramètres de l'opération peuvent ensuite être réglés à partir de la section principale qui apparaitra au centre de la fenêtre. Il est aussi possible de changer des paramètres de base vers le haut de la fenêtre, ou avec un clic-droit sur une opération dans las case Utilisées.

  • Quand les paramètres des opérations sont changés, leurs résultats sont calculés et affichés automatiquement. Si ce comportement n'est pas désirable, dé-cocher la case Prévisualisation automatique, et les changements seront alors affichés uniquement en cliquant le bouton Prévisualiser.

  • Dans le panneau Erreurs/Avertissements, les erreurs qui empêchent le renommage de s'effectuer sont affichées en rouge, les avertissements en jaune.
  • Si tout a l'air correct, cliquer le bouton Allons-y!. Votre sélection sera alors renommée.

  • Vous avez une possibilité de remettre la sélection à l'état initial une fois l'opération terminée.

La plupart des éléments du logiciel sont dotés d'infobulles d'aide si le pointeur de la souris est laissé au-dessus pendant quelques instants.
Les autres panneaux de cette fenêtre contiennent plus de détails sur toutes les opérations et options disponibles.
